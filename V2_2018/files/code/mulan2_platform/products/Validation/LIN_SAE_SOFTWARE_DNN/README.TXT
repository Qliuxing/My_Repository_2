Initial NAD : 0x6F
Supplier ID : 0x5AFE
Function ID : 0xFEED
Variant ID  : 0x01

Application's messages
----------------------
Number of messages per node : 4  (Frame IDs are assigned based on current DNN)

message 0 : slave-to-master, 2 bytes
message 1 : master-to-slave, 8 bytes
message 2 : not used (discarded)
message 3 : not used (discarded)


Diagnostic and Configuration messages :
- support Assign NAD request according to LIN2.0 specification
- support Assign Frame ID request according to LIN2.0 specification
- support Read by Identifier according to LIN2.0 specification
